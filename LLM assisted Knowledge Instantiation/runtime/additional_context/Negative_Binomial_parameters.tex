\maketitle

\textbf{Properties:}

1. The expected value of \( X \) is given by:
   \[
   \mathbb{E}(X) = \frac{r(1 - p)}{p}
   \]

2. The variance of \( X \) is given by:
   \[
   \text{Var}(X) = \frac{r(1 - p)}{p^2}
   \]

\textbf{Depends\_on:} \\
Binomial Distribution, Bernoulli Trials
