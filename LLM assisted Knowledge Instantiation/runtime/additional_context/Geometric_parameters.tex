\section{Parameters}

1. The expected value of \( X \) is given by:
   \[
   \mathbb{E}(X) = \frac{1}{p}
   \]

2. The variance of \( X \) is given by:
   \[
   \text{Var}(X) = \frac{1 - p}{p^2}
   \]
